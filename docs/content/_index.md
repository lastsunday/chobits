## 目的

- 学习 Rust 编程语言、语音交互和大语言模型技术。
- 构建一个完全自托管的聊天机器人（包含 LLM、TTS 等全部组件），类似 [xiaozhi-esp32](https://github.com/78/xiaozhi-esp32) 的自托管服务端方案。
